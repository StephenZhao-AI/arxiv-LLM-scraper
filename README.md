# arXiv LLM Scraper

## Overview
An automated AI agent to retrieve and classify medical research papers from arXiv using LLMs.

## Features
- arXiv API integration for paper retrieval
- LLM-based classification of domain relevance
- Automated filtering
- Datasets & Images Scraper / Classifier

## Tech Stack
Python, requests, LLM APIs

## Notes
Designed to improve dataset diversity and reduce domain bias in downstream AI models
